◆ Update()

void Hand.Update ( )
inlineprivate

The function is called every frame. It checks if the hand is open or closed, and if it is open, it checks if the hand is moving. If the hand is moving, it plays the animation for the hand moving. If the hand is not moving, it plays the animation for the hand being open. If the hand is closed, it plays the animation for the hand being closed
